Реализовать функционал добавления, редактирования и удаления пользователей в списке с использованием Redux и хуков `useDispatch` и `useSelector`.

1.	Создайте новый React-проект.

Используйте `vite` для создания нового проекта.

Перейдите в созданный проект.

2.	Установите необходимые библиотеки.

Установите `redux`, `@reduxjs/toolkit` и `react-redux`.

3.	Создайте хранилище (store) Redux.

Создайте файл `store.js`.

Настройте и экспортируйте хранилище Redux, используя `configureStore` из `@reduxjs/toolkit`.

4.	Создайте срез состояния для пользователей.

Создайте файл `userSlice.js`.

Определите начальное состояние, содержащее массив пользователей.

Используйте `createSlice` для создания среза состояния с именем `users`, определите редюсеры для добавления, редактирования и удаления пользователей, и экспортируйте действия и редюсер.

5.	Подключите хранилище к вашему приложению.

В корневом компоненте `App.js` оберните ваше приложение в `Provider` из `react-redux` и передайте ему хранилище.

6.	Создайте компонент для отображения списка пользователей и формы для добавления/редактирования пользователей.

Создайте файл `UserList.js`.

Используйте `useSelector` для извлечения списка пользователей из состояния.

Отобразите список пользователей в виде таблицы или списка.

Добавьте форму с полями ввода для имени, возраста, и электронной почты, а также кнопками для добавления нового пользователя и редактирования существующего.

Используйте `useDispatch` для отправки действий добавления и редактирования пользователей.

7.	Создайте компонент для отображения отдельного пользователя с возможностью удаления.

Создайте файл `UserItem.js`.

Отобразите информацию о пользователе (имя, возраст, электронная почта) и кнопку "Удалить".

Используйте `useDispatch` для отправки действия удаления пользователя.

8.	Проверка

Запустите приложение и убедитесь, что вы можете добавлять новых пользователей, заполняя поля ввода и нажимая кнопку "Добавить".

Убедитесь, что вы можете редактировать существующих пользователей, нажимая кнопку "Редактировать" и внося изменения в поля ввода.

Убедитесь, что вы можете удалять пользователей, нажимая кнопку "Удалить" рядом с каждым пользователем.

Убедитесь, что состояние пользователей корректно обновляется в хранилище Redux и интерфейс отображает актуальные данные.

*Дополнительно

Добавьте валидацию для формы добавления/редактирования пользователей (например, проверку на корректность введенной электронной почты).

Добавьте фильтрацию и сортировку списка пользователей.
